# Baseline Inventory

## Purpose

This document captures the known hardware available at the beginning of the CMR Lab 2026 project.

It serves as a historical reference point and should not be considered the current asset register.

The current state of the environment is maintained in:

```text
docs/02-asset-register.md
```

The purpose of this document is to show the starting position of the project and to demonstrate how the lab evolved over time.

## Observations

At the start of the project, the environment consisted primarily of repurposed equipment, legacy hardware and devices accumulated over time.

The initial challenge was not acquiring additional equipment but understanding what already existed, validating hardware capabilities and creating a structured plan for rebuilding the lab.

This inventory formed the basis for subsequent hardware validation, architecture planning and service deployment activities documented throughout the repository.
